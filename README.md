# soldity-starter

solidity 学习项目。

在线文档：[https://doc.yoouu.cn/blockchain/solidity.html](https://doc.yoouu.cn/blockchain/solidity.html)

## 📌 快速上手

**安装依赖**

```shell
yarn
```

**运行**

进入到 packages 执行目录下 `package.json` 内的脚本，例如

```shell
cd .\packages\vue3-client-hardhat\

yarn dev
```
