## 📌 项目玩法收集



### Shark

SSS - 治理代币

- 提案

SEA - 游戏代币

- 通过控制燃烧比调节平衡，初始流通 1 亿枚
- 游戏的鲨鱼 nft 只能用 sea 从官方商城购买盲盒，购买盲盒花费和饲养鲨鱼消耗的 sea 会被烧掉，保证了 sea 的内耗



游戏运行

1. 盲盒空投
2. 盲盒出售
3. 打开盲盒获得 shark
4. 上线 sea 代币
5. shark 战斗 获得 sea
6. 使用 sea 升级你的鲨鱼
7. 使用 sea 购买鲨鱼
